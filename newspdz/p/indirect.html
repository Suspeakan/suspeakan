ХАЙЛОЦКАДУХА: ШАЙ

The Indirect Object: Shaj

хүп тихё номдоликызатжи:
Suspeakan
корам демизопызатҳой:
2023-09-26



In СЫПЕҖ, the indirect object (called the хайлоцкадух, xajlockadux) is handled with a particle, which is placed before the direct object. For example, the sentence "Йо̄ ҳарлэтызат йе шай илинцы яшдо"

/ Йо̄ ҳарлэтызат¹ йе шай илинцы яшдо 1sg give¹-PST 2sg INDOBJ day-PL four // ¹ "give", "supply", or "provide" assumes some sort of exchange, whereas демизоп does not.

шай connects the indirect object [phrase] to the object [phrase]; here, шай connects йе (you), to илинцы яшдо (four days, ilincí jashdè). You are the recipient of four days, which I gave you

Another example: "Тамой демизопызат йейны попиленщҳой шай нэц приҗимиҳой-ко"

/ Тамой демизопызат йейны попиленщҳой шай {нэц приҗимиҳой-ко}ₐ 3sg give¹-PST woman cry-PTCPL INDOBJ {PTCLPN₁ hug-PTCPLN₂}(hug)ₐ

The indirect object phrase can also be put inside a mobile abverbial phrase! Take a modifaction of the first example sentence, "У шай илинцы яшдо да, йо̄ ҳарлэтызат йе." The MADVP is, as we know, anchored to the main verb, and modifies the main verb in whole, which isn't the case if you just string together adverbs; so, it should make very little sense or be straight-up ungrammatical to include the indirect object in the MADVP, but no, it is the exception. The MADVP, true to it's name, is mobile, and for only the indirect object it is a "bus" of sorts, the "INDOBJ bus", which allows for different ways to say the same sentence, so that one can emphasize the direct object, or be poetic, for example.

So the modified first example, with the INDOBJ bus, could be translated like so: "Four days, I gave you [four days]", emphasizing the "four days".
